Здравствуйте! 

Вас приветствует плагин для AutoCAD, предназначенный для оценки точности получения координат марок при построении сети в виде линейной пространственной засечки.

Вот действия, которые необходимо совершить для начала работы с программой:
1. Запустить AutoCAD

2.а Открыть пустой чертеж, если Вы планируете загружать исходные данные из файла Excel
Важно: данные в файле должны быть представлены в следующем формате:

Имя       |X             |Y             |Z
P1        | 1.123        | 1.123        | 1.123
1         | 0            | 0            | 0

2.б Открыть чертеж с моделью, которую необходимо проанализировать
Важно: каждая марка в файле должна быть представлена примитивом точки, имеющим соответствующие координаты, и текстом с ее именем рядом с точкой. Координаты текста должны совпадать с координатами точки.

3. Далее необходимо ввести в командной строке команду NETLOAD, позволяющую загрузить плагин в AutoCAD. Откроется окно выбора файла, найдите файл под названием MarkAnalyzerAutocadPlagin.dll и загрузите его. Возможно после этого AutoCAD спросит Вас, действительно стоит ли загружать данный файл, ведь он от неизвестного разработчика. Но я, как разработчик, обещаю, что ничего страшного не произойдет, если Вы загрузите.

4. Далее появится окно приветствия, которое расскажет, что делать дальше. 

Приятной работы!
